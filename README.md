Мощный VPN-клиент для Android с поддержкой множества протоколов, цепочек прокси и гибкой маршрутизацией.

## ✨ Возможности

### 📡 Поддерживаемые протоколы
- ✅ **WireGuard** — современный и быстрый VPN-протокол
- ✅ **Xray** — VMess, VLESS, Trojan, Shadowsocks, SOCKS5, HTTP/HTTPS
- ✅ **Tor** — анонимная сеть (встраивается как библиотека)
- ✅ **OpenVPN** — классический протокол
- ✅ **MTProto** — прокси для Telegram
- ✅ **DNS-over-HTTPS / DNSTT** — обход блокировок через DNS

### 🔗 Цепочки прокси
- Создание последовательных цепочек из любых протоколов
- Пример: `WireGuard → Tor → Выход`
- Пример: `Xray (VLESS) → Shadowsocks → Интернет`

### 🎯 Правила маршрутизации (Split Tunneling)
- По приложениям (Telegram → MTProto)
- По доменам (openai.com → Tor)
- По IP-диапазонам
- Правило по умолчанию

### 📤 Шеринг и импорт
- Поделиться конфигом через QR-код
- Импорт конфигов из файлов (.conf, .ovpn, JSON)
- Экспорт ваших настроек

### ⚙️ Режимы работы
- **TUN-режим** — системный VPN для всех приложений
- **Прокси-режим** — локальный SOCKS5/HTTP прокси
