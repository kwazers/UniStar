**UniStar** — это высокопроизводительный VPN-клиент для Android, объединяющий мощь лучших сетевых протоколов. Благодаря гибкой системе цепочек прокси и продвинутой маршрутизации, приложение позволяет обходить самые строгие сетевые ограничения, сохраняя при этом приватность и высокую скорость.

---

## ✨ Основные возможности

### 📡 Поддерживаемые протоколы
* **WireGuard**: Максимальная производительность и энергоэффективность.
* **Xray (Core)**: Поддержка VLESS, VMess, Trojan, Shadowsocks, SOCKS, HTTP.
* **Tor**: Глубокая анонимизация трафика через луковую сеть.
* **OpenVPN**: Стандарт индустрии с широкой поддержкой серверов.
* **MTProto**: Оптимизированный прокси для Telegram.
* **DNS Security**: Защита от DNS-инъекций через DoH (DNS-over-HTTPS) и DNSTT.

### 🔗 Цепочки прокси (Proxy Chaining)
Создавайте сложные маршруты для трафика, последовательно объединяя протоколы:
* `WireGuard → Tor` (Конфиденциальность в защищенном туннеле)
* `Xray (VLESS) → Shadowsocks` (Обход продвинутых систем DPI)

### 🎯 Интеллектуальная маршрутизация
* **Split Tunneling**: Указывайте, какие приложения используют VPN, а какие — прямой доступ.
* **Правила на основе доменов/IP**: Автоматическое переключение прокси при посещении специфических ресурсов.

### ⚙️ Режимы работы
* **TUN-режим**: Системный VPN, охватывающий весь исходящий трафик устройства.
* **Прокси-режим**: Поднятие локального SOCKS5/HTTP-сервера для использования в отдельных приложениях.
